Instumentation Testing

Pengujian terhadap activity main untuk memastikan apakah:

1. Menampilkan data stories
    - Memastikan rcy_stories dalam keadaan tampil
    - Gulir rcy_stories ke posisi data terakhir halaman pertama yaitu 10
    - Gulir lagi rcy_stories ke posisi terakhir data halaman kedua yaitu 20
2. Menampilkan data detail story
    - Memberikan tindakan klik pada data pertama rcy_stories
    - Memastikan TextView tv_detail_description dalam keadaan tampil

3. Menampilkan halaman utama melalui login
    - Memberikan tindakan pada Edit Text email dengan user yang sesuai
    - Memberikan tindakan pada Edit Text password dengan password yang sesuai
    - Memberikan tindakan klik pada button login
    - Memastikan berhasil pindah halaman lain dengan apakah btn_story visibl
4. Cek gagal ketika login
    - Memberikan tindakan pada Edit Text email dengan user yang sesuai
    - Memberikan tindakan pada Edit Text password dengan password yang tidak sesuai
    - Memberikan tindakan klik pada button login
    - Memastikan gagal login dengan cek apakah EditText email terlihat
